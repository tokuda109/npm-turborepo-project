# npm-turborepo-project

## Getting started

```
$ npm i
```

## Commands

### App

#### Build / Generate

```
$ turbo run app#build
$ turbo run app#generate
```

#### Dev / Start

```
$ turbo run app#dev
$ turbo run app#start
```

### E2E

```
$ turbo run e2e#run
```

### Lint

Run lint script across all workspaces:

```
$ turbo run lint
```
